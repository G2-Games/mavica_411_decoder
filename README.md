# Sony Mavica `.411` Thumbnail Decoder
This is a decoder for Sony Mavica .411 thumbnails.

The binary is very easy to use. Running the following:
```
mavica_411 MVC-654F.411
```

Will produce a file `MVC-654F.411.BMP`.
